# SenseLink 🚀
### An Inclusive Adventure for Every Unique Mind

SenseLink is a specialized educational and play platform designed specifically for neurodivergent children. It adapts its entire interface, interaction model, and sensory output to meet the unique needs of children with Autism, ADHD, and AuDHD.

---

## ✨ Core Design Philosophy

SenseLink isn't just one app—it's three experiences in one. We believe that a "one size fits all" approach doesn't work for neurodivergent kids.

### 🧩 Autism Mode (Calm)
*   **Visuals:** Soft, pastel color palettes (blues, greens, soft yellows) and a clean "dots" background pattern.
*   **Interaction:** Predictable, structured, and focused. Minimal distractions and slow, rhythmic animations.
*   **Focus:** Routine building, emotion matching, and shape sorting.

### ⚡ ADHD Mode (Active)
*   **Visuals:** High-contrast, energetic colors (orange, teal, red) with twinkling star overlays.
*   **Interaction:** High feedback, fast-paced, and novelty-driven. Includes "Turbo Mode" and expressive motion.
*   **Focus:** Immediate rewards, speed challenges, and high-stimulus play (Pet Popper, Rocket Blast).

### 🌌 AuDHD Mode (Mixed)
*   **Visuals:** Vibrant purple and indigo gradients.
*   **Interaction:** A balanced cocktail of stimulation. Provides a "Mode Toggle" (Sun/Moon) so the child can switch between Calm and Active states depending on their immediate sensory needs.
*   **Focus:** Creative building and pattern discovery (Gravity Blocks, Star Builder).

---

## 🎮 Features

### 🕹️ The Play Zone
A collection of custom-built mini-games:
*   **Sensory Sand:** A relaxing drawing experience for sensory regulation.
*   **Daily Routine:** Visual sequencing to help understand transitions.
*   **Rapid Tap & Mole Mash:** Engaging physical coordination challenges for ADHD energy.
*   **Gravity Blocks:** Physics-based building that is both stimulating and orderly.

### 📚 Story World & Discovery
*   **Curated Libraries:** Hand-picked, high-quality YouTube content featuring favorites like *Pete the Cat*.
*   **Interactive Quizzes:** "Hotwords" in descriptions trigger mini-quizzes to check comprehension in a fun way.

### 🏆 Reward System
*   **Stars:** Earned for watching videos and completing sections.
*   **Trophy Room:** A dedicated space showing collected badges (e.g., "Sand Artist", "Turbo Fingers").

## 🚀 Getting Started

### Prerequisites
*   Node.js (LTS version recommended)
*   npm or yarn

### Installation
1.  Clone the repository
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

### Building for Production
```bash
npm run build
```

## 💡 Accessibility Note
SenseLink uses a **Glassmorphism** UI style combined with **Bento Grid** layouts to ensure that while the interface is "attractive," it remains clear, high-contrast, and easy to navigate without overwhelming the user's cognitive load.
